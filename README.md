## 使い方

> [PHP 8.1+](https://php.net/releases/) が必要です

はじめに、はてなブログに接続するために認証情報を[詳細設定](https://blog.hatena.ne.jp/my/config/detail)から取得してください。

```bash
composer require toarupg0318/hatena-blog-client
```
